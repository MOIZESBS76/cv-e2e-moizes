# 📌 CV E2E — Projeto de Currículo Técnico Online

Este repositório contém o desenvolvimento do **Currículo Técnico Online E2E**, estruturado com foco em organização, clareza e boas práticas de documentação profissional. O objetivo deste projeto é demonstrar competências técnicas relevantes para o mercado, como:

- Levantamento e análise de requisitos  
- Documentação UML  
- Organização de tarefas via Jira  
- Estruturação profissional de repositório  
- Preparação para evolução em API, Banco de Dados, Docker e Integração com Power BI
